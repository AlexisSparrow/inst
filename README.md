# DiplomaPraktikum
Дипломная работа для ЯндексПрактикум
#
Инструкция по установке и запуску:  
1.Скачать и распаковать архив   
2.Установить пакеты NPM с помощью GIT BASH (команда - npm install)  
3.Использовать команду Npm run dev
